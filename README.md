# agentic-digests

Archive of weekly AI research digests generated by the [last30days](https://github.com/mvanhorn/last30days-skill) research engine.

## Digests

- `dtc-marketing/` — DTC & ecommerce marketing signals (weekly)
- `ai-marketing/` — AI marketing tools & tactics (weekly)
- `ai-ecosystem/` — Claude Code, MCP, AI dev tooling (weekly)

Each file is the synthesized markdown sent to subscribers that week.
